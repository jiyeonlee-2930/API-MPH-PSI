# API-MPH: Adaptive Private Set Intersection with Minimal Perfect Hashing

## Overview
This repository contains the implementation code for the paper:
"Adaptive Private Set Intersection with Minimal Perfect Hashing 
for Privacy-Preserving Medical Data Computation"

## Requirements
- Python 3.9.15
- Ubuntu 24

## Installation
pip install -r requirements.txt

## Usage
python generate_synthetic_medical_data.py

## Dataset Generation
- Seed: 42
- Korean healthcare data standards (KCD-8, ICD-10 compatible)
- HL7 FHIR compatible
